# NeuropsyOL.github.io

Landing page for the NeuropsyOL mobile neuroscience app suite:
**RECORDA**, **SENDA**, and **VIEWA**.

Hosted at **https://neuropsyol.github.io** via GitHub Pages.

## Apps

| App | Description | Repo |
|-----|-------------|------|
| **RECORDA** | Android LabRecorder — records LSL streams to XDF | [NeuropsyOL/RECORDA](https://github.com/NeuropsyOL/RECORDA) |
| **SENDA** | Streams Android phone sensors as LSL | [NeuropsyOL/SENDA](https://github.com/NeuropsyOL/SENDA) |
| **VIEWA** | Visual event markers / stimuli with LSL output | [NeuropsyOL/VIEWA](https://github.com/NeuropsyOL/VIEWA) |

## Editing the page

The entire site is a single `index.html` file — edit it directly.
GitHub Pages serves the `main` branch automatically once enabled in the repo settings.
